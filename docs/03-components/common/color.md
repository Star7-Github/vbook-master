蓝河应用支持[颜色值类型](</component/common/common-styles>)

开发者可参考 [MDN 文档](<https://developer.mozilla.org/zh-CN/docs/Web/CSS/color_value>) 了解更多颜色值的入门知识

## 颜色值格式示例

  * `'#f0f'` (#rgb)
  * `'#ff00ff'` (#rrggbb)
  * `'#ff00ff13'` (#rrggbbaa)
  * `rgb(255, 0, 255)`
  * `rgba(255, 255, 255, 1.0)`
  * `black,white`
